The `njuvisual` package
=======================

The `njuvisual` package provides an interface to display logos related to Nanjing University.

Basic usage
-----------

    \njuemblem{3cm}{!}
    \njuname{3cm}{!}
    \njumotto{!}{3cm}
    \njuspirit{!}{3cm}

More information can be found in the package documentation.
